# Sector Arcade 🕹️

**Free browser arcade games — no download, no signup, play instantly.**

🌐 Live site: [sectorplanton.github.io/sector-games](https://sectorplanton.github.io/sector-games)

## Games
- 🧱 Sector Arkanoid — brick breaker with power-ups
- 🚀 Galaxy Assault — space shooter
- 🐦 Void Bird — one-tap flappy challenge
- 🏰 Castle Destroyer — physics siege game
- 🟪 Neon Blocks — neon puzzle (tetris-style)
- 🐍 Serpent Ultra — snake with power-ups

## Features
- ⚡ Instant play, no install
- 🔍 Game search in navbar
- 🎲 Random game button
- ♥ Favorites system (localStorage)
- 📱 Fully responsive (mobile/tablet/desktop)
- 📝 SEO blog with 5 articles
- 🗂️ 4 category pages
- 📊 Full sitemap.xml

## Setup
1. Clone/download this repo
2. Replace `YOUR_FORMSPREE_ID` in `contact.html` with your [Formspree](https://formspree.io) form ID
3. Add real affiliate links in `/blog/` articles (search for `href="#"`)
4. Deploy to GitHub Pages (Settings → Pages → main branch → root)
5. Submit `sitemap.xml` to Google Search Console

## Structure
```

/games/         → 6 game pages (SEO-optimized)
/blog/          → Blog index + 5 articles
/categories/    → 4 category pages
/icons/         → PWA icons
```
